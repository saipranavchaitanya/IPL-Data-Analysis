# IPL Data Analysis using Python and Power BI

## Project Overview
This project analyzes IPL match data to understand team performance,
season-wise trends, and venue analysis using Python and Power BI.

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- Power BI
- GitHub

## Dataset
- matches.csv
-deliveries.csv


## Key Analysis
- Total matches, teams, and seasons
- Team-wise wins analysis
- Matches per season trend
- Venue-wise match distribution

## Dashboard
Power BI dashboard includes KPI cards, bar charts, and line charts.

## Conclusion
This project demonstrates data cleaning, exploratory data analysis,
and interactive dashboard creation using real-world sports data.
